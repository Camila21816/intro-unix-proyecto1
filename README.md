Project description: Implementation of a secure Debian 13 environment using disk encryption (LUKS/LVM), advanced entropy-based password management, and integrity auditing with hash functions. The project incorporates the use of GPG to secure communications and ensure the authenticity of the development process through signed commits in Git, in compliance with professional cybersecurity and systems administration standards.


This project aligns with the following course objectives. Upon completion, students will have demonstrated practical proficiency in:

Describing the UNIX/Linux operating system and comparing it to other operating systems.

Becoming familiar with the UNIX/Linux file system, file attributes, and permissions.

Using GNU Privacy Guard (GPG) to encrypt, decrypt, and sign files and emails.

Demonstrating proficiency in essential UNIX/Linux commands from the terminal.

Understanding security fundamentals: disk encryption, password management, and hashing.

Managing repositories with Git and collaborating on platforms such as GitHub and Codeberg.
